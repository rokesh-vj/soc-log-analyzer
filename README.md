# SOC Log Analyzer & Threat Detection Tool

This project simulates SOC monitoring by analyzing system logs and detecting suspicious activity.

## Features
- Detects brute force login attempts
- Detects suspicious command execution
- Generates SOC alert reports
- Maps attacks to MITRE ATT&CK techniques

## Technologies Used
Python
Linux
Regex
MITRE ATT&CK

## Run the tool

python3 analyzer.py
